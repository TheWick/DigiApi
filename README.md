📱 DigiApi

Aplicación Android desarrollada en Kotlin que consume una API para mostrar información de Digimon de manera dinámica.

---

🚀 Características

- 🔍 Consumo de API REST
- 📋 Listado de Digimon
- 📄 Vista de detalle de cada Digimon
- ⚡ Arquitectura basada en ViewModel
- 🔄 Manejo de datos asíncronos

---

🛠️ Tecnologías usadas

- 🟣 Kotlin
- 🏗️ MVVM (Model - View - ViewModel)
- 🌐 Retrofit (consumo de API)
- 🔄 Coroutines
- 🧩 LiveData / StateFlow

---


---

⚙️ Instalación

1. Clona el repositorio:

git clone https://github.com/TheWick/DigiApi.git

2. Abre el proyecto en Android Studio

3. Ejecuta la app en un emulador o dispositivo físico

---

📡 API utilizada

Este proyecto consume una API pública de Digimon.

---

📂 Estructura del proyecto

com.example.digiapi
│
├── data        # Lógica de datos y API
├── viewmodel   # ViewModels
├── ui          # Pantallas (Activities/Fragments)
